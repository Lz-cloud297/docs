## API 通用描述
所有请求参数必须组装成 **JSON 格式** 的请求体，并使用 **RSA 公钥** 对整个 JSON 请求体加密。将加密后的数据进行 **Base64 编码**，然后将 Base64 编码后的数据作为 `data` 参数提交给 API 接口。

## 钱包地址格式
- **TRON**: 地址中字母区分大小写。
- **ETH**: 地址中字母统一为小写。

## 风险评分参考
- **低风险**: 0 ~ 30，无风险或低风险。
- **中风险**: 31 ~ 64，可能与中高风险地址有过交易。
- **高风险**: 65 ~ 84，与中高风险地址有过交易，建议隔离。
- **严重风险**: 85 ~ 100，与高危地址有交易，建议立即隔离。
